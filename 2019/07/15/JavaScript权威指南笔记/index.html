发表于 2019-07-15 | 分类于 杂项 ， 总结

JavaScript权威指南随手整理一些笔记

数据类型

原始类型：Number，String，Boolean

特殊值：null，undefined

Object

ES6新增：Symbol

垃圾回收

JS解释器的内存管理机制可以自动对内存进行垃圾回收。这意味着程序可以按需创建对象，而不必对这些对象进行销毁和内存回收。当不再有任何引用指向一个对象，解释器就会知道这个对象没用了，然后自动回收它所占用的内存资源

常用算数运算函数
NaN

与任何值不相等，包括自身

二进制浮点数

JavaScript采用IEEE-745浮点数表示法，二进制表示法，可以精确的表示诸如1/2，1/2048等，但并不能精确表示十进制分数如1/10

字符串

JS中的字符串是固定不变的，类似replace()和toUpperCase()方法都返回新的字符串，原字符串本身并没有发生变化

实际上原始值（undefined，null，Boolean，number，string）均是不可更改的

null和undefined

null可以被认为是一个特殊的对象值，但通常认为它是自有类型的唯一成员

数组转换为数字

空数组被转换为数字0以及具有单个元素的数组同样被转换为一个数字的原因：

数组继承默认的valueOf()方法，这个方法返回一个对象而不是原始值，因此，数组到数字的转换调用toString()方法

空数组转换为空字符串，空字符串转换为数字0

含有一个数字元素的数组转为字符串，再转换为数字

类型转化

JS语言核心的内置类首先尝试使用valueOf()，再尝试使用toString()，除了日期类，日期类只使用toString()

日期类是JavaScript语言核心中唯一的预先定义类型，它定义了有意义的向字符串和数字类型的转换

作用域

函数体内，局部变量的优先级高于同名的全局变量

JS函数作用域：变量在声明它们的函数体以及这个函数体嵌套的任意函数体内都是由定义的

当通过var声明一个变量时，这个变量无法被delete删除

作用域链：每一段JS代码都有一个与之关联的作用域链。这个作用域链是一个对象列表或者链表，这组对象定义了这段代码”作用域中“的变量。

当JS需要查找变量x的时候（变量解析），它会从链中的第一个对对象开始查找，如果这个对象有一个名为x的属性，则会直接使用这个属性的值，如果第一个对象中不存在名为x的属性，JS会继续查找链上的下一个对象，以此类推，如果最终不存在x则抛出一个引用错误（ReferenceError）异常

对象的比较

JS中对象的比较是引用的比较，而不是值的比较

对象和其本身是相等的，但和其它任何对象不相等

如果两个不同的对象具有相同数量的属性，相同的属性名和值，他们依然是不相等的

相等位置的数组元素是相等的两个数组也是不相等的

ASCⅡ字母大小

所有的ASCⅡ字母都”小于“小写的ASCⅡ字母

eval()

JS通过全局函数eval()完成“解释运行由js源代码组成的字符串，并产生一个值”这个工作

PS：一般来讲，如果一个函数调用了eval()，那么解释器将无法对这个函数做进一步的优化

eval()只有一个参数，如果传入参数不是字符串，则直接返回这个参数；如果参数是字符串，则会对这个字符串进行编译，编译失败则抛出一个语法错误（SyntaxError）异常。如果编译成功则开始执行这段代码并返回字符串中的最后一个表达式或语句的值，如果最后一个表达式或者语句没有值，则最终返回undefined

严格模式禁用with（ES6将整个语言提升至严格模式），非严格模式也不建议使用

严格模式和非严格模式的部分区别
  • with语句
  • 严格模式中，所有的变量都要先声明，否则会抛出引用异常错误
  • 严格模式中，调用的函数（不是方法）中的this值是undefined
  • 严格模式中，当通过call()或apply()调用函数时，其中this值就是通过call()或apply()传入的第一个参数，非严格模式中，null和undefined值被全局对象转换为对象的非对象值所代替
new

通过new运算符创建并初始化一个对象，关键字new后跟随一个函数调用，这里的函数称作构造函数，构造函数用以初始化一个新创建的对象

对象及对象原型

每一个JS对象（null除外）都和另一个对象相关联。“另一个”对象就是我们熟知的原型，每一个对象都从原型继承属性

可以通过Object.prototype获得对原型对象的引用

没有原型的对象不多，Object.prototype就是其中之一，他不继承任何属性。其他原型对象都是普通对象，普通对象都具有原型。所有的内置构造函数（以及大部分的自定义构造函数）都具有一个继承自Object.prototype的原型。例如：

Date.prototype的属性继承自Object.prototype，因此由new Date()创建的Date对象的属性同时继承自Date.prototype和Object.prototype。这一系列链接的原型对象就是所谓的“原型链”

如果一个对象的原型是保留字，就必须通过方括号的形式访问他们

JS的对象都是关联数组

Object.creat()

Object.creat()创建一个新对象，其中第一个参数是这个对象的原型。Object.creat()提供第二个可选参数，用以对象的属性进行进一步的描述

inherit()函数的其中的一个用途就是防止库函数无意间修改那些不受控制的对象
